## Temperature Converter
<img width="1920"  alt="image" src="https://github.com/user-attachments/assets/8748b388-51ec-4d3b-80cb-4cdb9515d18f" />

This JavaScript project is a simple, single-function temperature conversion tool that converts an input value from Celsius to Fahrenheit. An event listener is attached to a "yell" button, which, when clicked, triggers the convert function. The function retrieves the numeric value entered by the user in an input field (#sel), applies the standard conversion formula (tem * 9/5 + 32), and then dynamically updates a designated DOM element (#placeToYell) to display the resulting temperature in Fahrenheit.
